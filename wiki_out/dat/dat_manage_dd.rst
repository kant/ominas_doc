 dat_manage_dd, dd



Description
-----------
	Adds a data descriptor to the NV state maintained list.  If the list
	is full, the oldest descriptor is unloaded and removed.













+++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++


Parameters
----------




dd
-----------------------------------------------------------------------------

*in* 

Data descriptor.
